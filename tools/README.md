# 似たもの判別プロジェクト

総合探究用 - AIと人間で似たものを判別するソフトウェア

## 概要

このプロジェクトは、双子や似た動物など、非常に似たものをAIと人間に判断させて比較するためのソフトウェアです。

## セットアップ

### 1. 必要なライブラリをインストール

```bash
pip install -r requirements.txt
```

### 2. 画像をダウンロード

```bash
python image_downloader.py
```

## 機能

### 画像ダウンローダー (`image_downloader.py`)

- **モード1**: フォルダ作成のみ
- **モード2**: URLを手動入力してダウンロード
- **モード3**: 似たもの画像を自動検索してダウンロード
- **モード4**: カスタム検索

### 似たものの例

| カテゴリ | 例 |
|---------|-----|
| 動物 | チーター vs ヒョウ |
| 動物 | アライグマ vs タヌキ |
| 動物 | アシカ vs アザラシ |
| 犬種 | 柴犬 vs 秋田犬 |
| 鳥 | カラス vs ワタリガラス |

## フォルダ構造

```
downloaded_images/
├── twins/           # 双子の画像
├── similar_animals/ # 似た動物
├── similar_dogs/    # 似た犬種
├── similar_cats/    # 似た猫種
├── similar_birds/   # 似た鳥
└── test_images/     # テスト用
```

## 著作権について

⚠️ **重要**: ダウンロードした画像の著作権に注意してください。

- 教育目的での使用でも、出典を明記することをお勧めします
- 商用利用は避けてください
- 可能であれば、著作権フリーの画像を使用してください

### 著作権フリー画像サイト

- [Unsplash](https://unsplash.com/)
- [Pexels](https://www.pexels.com/)
- [Pixabay](https://pixabay.com/)
